## 🎲 常用 DC 難度

| 難度 | DC | 說明 |
|------|----|----|
| 非常簡單 | 5 | 幾乎所有人都能做到 |
| 簡單 | 10 | 一般人經過努力可以做到 |
| 中等 | 15 | 需要一定技巧或訓練 |
| 困難 | 20 | 專業人士的挑戰 |
| 非常困難 | 25 | 大師級難度 |
| 幾乎不可能 | 30 | 傳奇挑戰 |

## 🎯 擲骰公式

### 技能檢定
```
d20 + 屬性調整值 + 熟練加值 (如有)
```

**成功率 ≥ 95%** (所需骰值 ≤ 2) → 自動成功,不擲骰

### 攻擊檢定
```
d20 + 屬性調整值 + 熟練加值 vs AC
```
- 自然 20 = 重擊 (傷害骰加倍)
- 自然 1 = 自動失敗

### 傷害
```
武器骰 + 屬性調整值
```
- 重擊 = 武器骰 × 2 + 屬性調整值

### 豁免檢定
```
d20 + 屬性調整值 + 熟練加值 (如有) vs DC
```

## ⚔️ 戰鬥速查

### 先攻
```
d20 + 敏捷調整值
```

### 每回合行動
- ✅ 1 個動作 (Action)
- ✅ 1 個附贈動作 (Bonus Action)
- ✅ 移動 (通常 30 呎)
- ✅ 1 個反應 (Reaction,在他人回合使用)

### 常見動作
- 攻擊
- 施法
- 衝刺 (移動距離加倍)
- 撤離 (移動不引發機會攻擊)
- 閃避 (攻擊者劣勢)
- 幫助 (給予盟友優勢)
- 躲藏
- 準備 (設定觸發條件)

### 機會攻擊
敵人離開你的觸及範圍時,可使用反應進行一次近戰攻擊

## 🏪 服務價格 (依當地經濟,首次生成後固定)

### 定價原則
- 住宿/餐飲 = 當地日薪的 10-30%
- 高級服務 = 當地日薪的 50-100%
- 基礎消耗品 = 當地日薪的 1-5%

### 新手村範例
- 一餐 (普通): 2-5 銅
- 住宿 (單人房): 1-3 銀/晚
- 麥酒: 5-10 銅
- 乾糧 (1天份): 3-5 銅

**重要:** 實際價格查 `data/prices.json`,首次使用時生成

## 🧙 法術

### 法術攻擊檢定
```
d20 + 施法屬性調整值 + 熟練加值
```

### 法術豁免 DC
```
8 + 施法屬性調整值 + 熟練加值
```

### 專注檢定
受傷時 DC = max(10, 傷害值 / 2)
