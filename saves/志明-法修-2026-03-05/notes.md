# GM 備註 - 志明

## 世界設定

### 靈根制度
- 單靈根：修煉速度最快，然天花板止步大羅金仙之前（除非有大毅力或過人天資）
- 雙靈根：速度較快，上限略高於單靈根
- 三靈根以上：速度漸慢，但天花板更高
- 五行靈根：世人眼中的廢材，修煉最慢（XP ×0.8）；實為最符合天道之根骨，潛力無上限，可突破至道祖甚至聖人之境

### 太乙門
- 玄黎大陸上的小門派，規模不大但團結一心
- 掌門與師父（外務長老兼丹藥管理人）為師兄弟關係
- 志明身世：嬰兒時父母意外雙亡，在樹林中被師父撿回，自幼在門派長大
- 志明同期有五位師兄弟，感情甚好
- 師父自幼教導志明丹藥知識，有意讓其在修煉之路上有完善的丹藥輔助

### 境界體系

#### 小世界（凡人界）
煉氣（1-9 層）→ 築基（初/中/後期）→ 結丹/金丹（初/中/後期）→ 元嬰（初/中/後期）→ 化神（初/中/後期）→ 煉虛（初/中/後期）→ 合體（初/中/後期）→ 大乘（初/中/後期）

- 大乘圓滿後，天道降下劫數（天劫）
- 渡劫成功 → 飛升大世界，蛻變為仙人

#### 大世界（仙界）
真仙 → 金仙 → 太乙金仙 → 大羅金仙 → 道祖 / 聖人（頂點）

- 道祖與聖人為超然存在，不受天道束縛

### 目前謎團
- 志明父母究竟出自哪個大門派？
- 當年追殺父母的勢力是否仍在追查？
- 玄黎大陸的靈氣為何近年濃度下降？
- 深海寒玉的製作技術極為罕見，父母究竟是何等修為？

## 主線伏筆

- 父母遭大門派追殺，敵人身份未明，威脅依然存在
- 玉佩若曝光，必招滅頂之災，太乙門無力庇護
- 志明的靈液品質遠超同階，築基後品階將極為驚人
- 師父年事已高（金丹期），未來能否持續庇護志明是隱憂
- 大師兄火屬性單靈根，進展極快，太乙門恐怕留不住他，離去後對門派影響未知
